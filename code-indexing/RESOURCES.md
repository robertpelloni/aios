# Code Indexing Resources

## Directory Structure

### AST (Abstract Syntax Trees)
- `ast/ast-grep` - AST-based code search and transformation tool
- `ast/tree-sitter` - Incremental parsing system for syntax highlighting and code analysis

### Tools
- `tools/aider` - AI pair programming in terminal (repo map, code context)
- `tools/bloop` - AI-powered code search engine with semantic understanding

### LSP (Language Server Protocol)
- `lsp/` - Language server implementations for IDE integration

### Grep Tools
- `grep/` - Advanced code search tools (ripgrep, ag, etc.)

## Capabilities

### Code Search
- Semantic code search across repositories
- AST-aware pattern matching
- Multi-language support
- Repository-wide refactoring

### Code Analysis
- Syntax tree parsing and traversal
- Code structure extraction
- Symbol resolution and navigation
- Cross-reference generation

### Integration Points
- IDE/editor integration via LSP
- CLI tools for terminal workflows
- API for programmatic access
- MCP server compatibility

## Use Cases
- Codebase understanding for AI agents
- Automated refactoring
- Code review assistance
- Documentation generation
- Dependency analysis
