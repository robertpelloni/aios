# Vision: The Bobcoin Economy

## Core Philosophy
Bobcoin is not just a cryptocurrency; it is a **Proof of Health** economy. It rewards physical activity (specifically dancing), social interaction, and healthy relationships. It is designed to be anti-hoarding and extremely high-velocity.

## Technical Pillars
1.  **Privacy:** Based on Monero/Solana tech stack for speed + anonymity.
2.  **Mining:** "Proof of Dance". Arcade machines and exercise equipment act as oracles that verify physical exertion and reward tokens.
3.  **Infrastructure:** These machines also double as:
    -   Tor Nodes
    -   Distributed Storage (MegaTorrents)
    -   Game Servers (MMORPG)

## The Super AI Plugin Role
This plugin acts as the software layer connecting the "Miners" (Arcade Machines/Users) to the Bobcoin Network.
-   **Core Service:** Manages the wallet and "Mining" sessions.
-   **Clients (Browser/CLI):** Act as the interface for the user to submit activity data or view their balance.
