# Database Resources

## Directory Structure

### MCP Servers
- `mcp-servers/dbhub` - Universal database MCP hub
- `mcp-servers/mysql` - MySQL database MCP server
- `mcp-servers/postgres` - PostgreSQL database MCP server

### Tools
- `tools/` - Database management and query tools

### Resources
- `resources/` - Schema definitions, migrations, templates

### Systems
- `systems/` - Complete database system integrations

## Capabilities

### Database Operations
- Query execution (SELECT, INSERT, UPDATE, DELETE)
- Schema introspection
- Transaction management
- Connection pooling

### Supported Databases
- PostgreSQL
- MySQL/MariaDB
- SQLite
- MongoDB (planned)
- Redis (planned)

### MCP Integration
- Natural language to SQL
- Schema-aware completions
- Query result formatting
- Error handling and recovery

## Use Cases
- AI-assisted database queries
- Automated schema management
- Data exploration and analysis
- Migration generation
- Database documentation
