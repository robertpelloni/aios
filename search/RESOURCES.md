# Search Resources

## Directory Structure

### MCP Servers
- `mcp-servers/web-search-mcp` - Web search MCP server
- `mcp-servers/mcp-everything-search` - Universal search MCP

### Tools
- `tools/DesktopCommanderMCP` - Desktop file and content search

## Capabilities

### Web Search
- Multi-engine search (Google, Bing, DuckDuckGo)
- Result parsing and summarization
- Image and news search
- Site-specific search

### Local Search
- File system indexing
- Content search (grep-style)
- Fuzzy matching
- Real-time updates

### MCP Integration
- Structured search results
- Pagination support
- Filter and sort options
- Caching for performance

## Use Cases
- AI agent information retrieval
- Document discovery
- Code search across repos
- Knowledge base queries
