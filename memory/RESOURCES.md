## Directory Structure

### Context Engineering
- `context-engineering/` - Context saving, loading, parsing, compaction techniques

### Session Management
- `session-management/` - Automatically saving session history, chats, export/import

### Memory Bridging
- `memory-bridging/` - Bridging memory between tools and systems

### History Parsing
- `history-parsing/` - Parsing and recognizing important vs unimportant details

### Compaction
- `compaction/` - Context compaction, removing unnecessary information

### Automatic Detection
- `automatic-detection/` - Auto-detection of session logs, format conversion

### Scope & Depth
- `scope-depth/` - Progressive loading, memory scoping, depth management

### Self-Improvement
- `self-improvement/` - Recursive self-improvement, self-learning systems

### Evolution
- `evolution/` - Evolving memory systems, adaptive memory

### Correction
- `correction/` - Memory correction, validation, consistency
