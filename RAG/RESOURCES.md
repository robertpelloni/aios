# RAG (Retrieval-Augmented Generation) Resources

## Directory Structure

### Methods
- `methods/` - RAG implementation methods and approaches

### Theories
- `theories/` - Theoretical foundations, research papers, concepts

### Types
- `types/` - Different RAG architectures and variations

### Systems
- `systems/` - Complete RAG system implementations

### Chunking
- `chunking/` - Document chunking strategies and algorithms

### Parsing
- `parsing/` - Document parsing, text extraction, preprocessing

### Import/Export
- `import-export/` - Data ingestion and export pipelines

### Documents
- `documents/` - Document processing, formats, handlers

### Research
- `research/` - RAG research, papers, benchmarks

## Topics Covered
- Document parsing and file handling
- Import/export pipelines
- Chunking strategies (fixed, semantic, recursive, etc.)
- Embedding methods and vector stores
- Retrieval strategies
- RAG architectures (naive, advanced, hybrid)
- Evaluation and benchmarking
- Multi-modal RAG
- Knowledge graph integration
