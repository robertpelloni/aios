# SuperAI CLI Resources

## Directory Structure

### Agents
- `agents/autogen` - Microsoft's multi-agent framework
- `agents/crewai` - Multi-agent orchestration framework
- `agents/smolagents` - HuggingFace's lightweight agents
- `agents/openhands` - AI coding agent (formerly OpenDevin)
- `agents/claude-squad` - Claude multi-agent coordination

### Routers
- `routers/claude-code-router` - Claude Code CLI routing
- `routers/gemini-cli-router` - Gemini CLI routing
- `routers/gemini-router` - Gemini API routing
- `routers/GoogleGeminiRouter` - Google Gemini integration

### Tools
- `tools/ccs` - Claude Code switcher
- `tools/cc-switch` - CLI switching utility
- `tools/Lynkr` - Tool linking utility
- `tools/code-assistant-manager` - Multi-assistant manager
- `tools/cc-switch-cli` - CLI-based assistant switching

### CLIs
- `clis/emdash` - Enhanced CLI interface
- `clis/CodeNomad` - Portable coding CLI
- `clis/claude-code-madapp` - Claude Code application

### Proxies
- `proxies/openai-gemini` - OpenAI to Gemini proxy
- `proxies/gemini-openai-proxy-*` - Various Gemini-OpenAI adapters
- `proxies/vercel-ai-proxy` - Vercel AI SDK proxy
- `proxies/ccproxy` - Claude Code proxy

### Extensions
- `extensions/gemini-superclaude-mcp-server` - Gemini + Claude MCP bridge

### Harnesses
- `harnesses/SuperClaude_Framework` - Claude orchestration framework

### Resources
- `resources/awesome-opencode` - Curated OpenCode resources
