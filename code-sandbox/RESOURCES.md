# Code Sandbox Resources

## Directory Structure

### Tools
- `tools/open-interpreter` - Natural language code execution
- `tools/e2b-code-interpreter` - Cloud-based code execution sandbox
- `tools/cohere-terrarium` - Isolated Python execution environment

### Runtimes
- `runtimes/dagger` - Programmable CI/CD engine with containerization

### MCP Servers
- `mcp-servers/` - Sandboxed code execution via MCP

## Capabilities

### Code Execution
- Multi-language support (Python, JavaScript, Bash, etc.)
- Isolated execution environments
- Resource limits and timeouts
- Output capture and streaming

### Security
- Container-based isolation
- Network restrictions
- Filesystem sandboxing
- Resource quotas (CPU, memory, disk)

### Integration
- REST API access
- MCP protocol support
- Streaming output
- File upload/download

## Use Cases
- AI-generated code execution
- Interactive coding assistants
- Automated testing pipelines
- Data analysis workflows
- Educational code runners
